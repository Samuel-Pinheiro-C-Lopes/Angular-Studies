Directives:

We have several directives - whose aplications are distinct - proving to be very useful. The first one I want to show is the *ngIf, wich works like an if statement, but changing the structure of the html component (that's the reason it takes * before it).

Paragraph bellow shall only be seen while the button is green, for example:

The button above is green!

We can also use a ng-template with an else statement to the ngIf directive to work with alternative text content, like bellow ("Right text" is printed as green, and every other content, red):

We can as well use the ngStyle and ngClass to work with style on our HTML elements. Using [ngStyle], we can set properties using functions exported in our components.ts, those functions shall return the value needed for the CSS property.

ngClass needs true or false, working as an object (like [ngStyle]) but with classes as keys, and booleans as values.
